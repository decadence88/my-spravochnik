## 💡 1. Блоки заметок и уведомлений (Admonitions)

*Используй их, чтобы структурировать информацию и выделять главное.*

!!! note "Полезный совет"
    Это стандартный блок для подсказок. Он аккуратный и не бросается в глаза, но отлично группирует мысли.

!!! info "Важная информация"
    Синий блок. Идеально подходит для описания того, как работает та или иная функция или фича.

!!! success "Решение найдено"
    Зеленый блок. Отлично подходит для фиксации успешных кейсов (например, как мы победили ошибку Vercel!).

!!! warning "Будьте внимательны"
    Желтый блок. Привлекает внимание к нюансам, где пользователь может случайно ошибиться.

!!! danger "Критическая ошибка / Опасность"
    Красный блок. Используй его для предупреждений, которые могут что-то сломать, если сделать неправильно.

## 📑 3. Переключаемые вкладки (Tabs)

*Шикарное решение для разделения инструкций по контексту (например, разные операционные системы или разные способы настройки).*

=== "🍏 Для macOS"
    ### Настройка на Маке
    Выполните команду в терминале:
    ```bash
    cd ~/Desktop/my-spravochnik && python3 -m mkdocs serve
    ```

=== "🪟 Для Windows"
    ### Настройка на Windows
    Откройте PowerShell и введите:
    ```powershell
    cd C:\Users\User\Desktop\my-spravochnik ; python3 -m mkdocs serve
    ```

=== "🐧 Для Linux"
    ### Настройка на Linux
    Используйте стандартный менеджер:
    ```bash
    python3 -m mkdocs serve
    ```
