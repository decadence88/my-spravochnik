=== "SUBSTR — Обрезка строк"

    <font size="5">**Вырезание части текста в PostgreSQL**</font>
    
    Функция `SUBSTR` (от англ. *substring*) позволяет отрезать от длинной строки нужный кусочек. Ты указываешь базе данных, с какого символа начать и сколько знаков вправо нужно забрать. Это незаменимо, когда нужно проверить конкретные цифры в системных кодах или номерах счетов.
    
    **Синтаксис функции:**
    `SUBSTR(что_режем, с_какого_символа, сколько_символов_взять)`
    
    *Важно:* В SQL отсчёт символов всегда начинается с **1**, а не с нуля.
    
    **Примеры базового синтаксиса:**
    
    * `SUBSTR('HELLO', 1, 2)` — начнёт с 1-го символа ('H') и возьмет 2 знака. Результат: `'HE'`.
    * `SUBSTR('2026-07-01', 6, 2)` — начнёт с 6-го символа и возьмет 2 знака. Результат (месяц): `'07'`.

    **💡 Живой пример в коде:**
    Отфильтруем счета так, чтобы в отчёт не попали филиалы с кодами '04' и '06' (код филиала зашит в номере подразделения `oe` с 6-го по 7-й символ):
    
    ```sql
    SELECT 
        p1.PERS_NR,
        p1.oe AS full_code,
        -- Показываем, какой именно кусочек мы вырезали
        SUBSTR(p1.oe, 6, 2) AS branch_code
    FROM S080TGL.T_PERSON p1
    -- Исключаем ненужные филиалы
    WHERE SUBSTR(p1.oe, 6, 2) NOT IN ('04', '06');
    ```

=== "EXTRACT — Работа с датами"
    <font size="5">**Извлечение компонентов из даты и времени**</font>

    Функция `EXTRACT` разбирает сложную дату (или штамп времени) на отдельные «запчасти». С её помощью можно легко вытащить только год, только месяц или только час, чтобы использовать их в математических расчётах, фильтрах `WHERE` или группировках.
    
    **Основные компоненты для извлечения:**
    `YEAR` (год), `MONTH` (месяц), `DAY` (день), `HOUR` (час), `MINUTE` (минута), `SECOND` (секунда).
    
    **Примеры базового синтаксиса:**
    
    * `EXTRACT(YEAR FROM '2026-02-23'::date)` — вытащит год числом: `2026`.
    * `EXTRACT(YEAR FROM CURRENT_DATE) + 1` — возьмёт текущий год и прибавит к нему единицу (математика).

    **💡 Живой пример в коде:**
    Посчитаем, сколько счетов было открыто в текущем году, сгруппировав их по месяцам (чтобы понять, в каком месяце клиенты были активнее всего):
    
    ```sql
    SELECT 
        -- Вытаскиваем только номер месяца (число от 1 до 12)
        EXTRACT(MONTH FROM GK.EROEFFNUNGSDATUM) AS open_month,
        COUNT(*) AS accounts_count
    FROM S080TGL.GRUNDKONTO GK
    -- Фильтруем, чтобы брать данные только за ТЕКУЩИЙ ГОД
    WHERE EXTRACT(YEAR FROM GK.EROEFFNUNGSDATUM) = EXTRACT(YEAR FROM CURRENT_DATE)
    GROUP BY EXTRACT(MONTH FROM GK.EROEFFNUNGSDATUM)
    ORDER BY open_month;
    ```
